Prerequisites
-------------
- Python 3.5. 
- node & npm: https://nodejs.org/
- make 

To install
----------

```
pip install -r requirements.txt
npm install coffee-script --save
```

To specify database directories
-------------------------------
Change the DB_DIR variable in the settings.py file.
